## Getting Started
---
#### Clone the repository and build the package.
- To run termiboard
```
git clone https://github.com/DarthBenro008/termiboard
cd termiboard
go run *.go
```
- To build termiboard 
```
git clone https://github.com/DarthBenro008/termiboard
cd termiboard
go build
./termiboard
```
